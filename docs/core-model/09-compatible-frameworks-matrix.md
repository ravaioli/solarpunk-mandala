## 3. Critical Integration Principles
1. **Ontological Integrity:** Any compatible framework must honor consciousness as primary (analytic idealism), not reduce to materialism
2. **Boundary Permeability:** Frameworks that harden boundaries (between mind/body, human/nature, spiritual/practical) require transformation before integration
3. **Temporal Wisdom:** Frameworks focused on short-term optimization rather than deep time wisdom need recontextualization
4. **Power Circulation:** Any framework that concentrates decision-making, resources, or meaning toward a center must be redesigned for circulation
5. **Embodied Grounding:** Abstract frameworks without embodied practices require grounding in the four foundations before integration
