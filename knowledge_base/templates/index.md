# Knowledge Base Templates Index

This folder contains markdown files used as templates for creating new content within the Solarpunk Mandala knowledge base.

To use a template, copy the content into a new file and fill in the bracketed placeholder text (`[... ]`).

| Template File | Purpose |
| :--- | :--- |
| [Archetype_Template.md](./Archetype_Template.md) | Blueprint for documenting an individual Archetype's full narrative, history, and applications. |
| [Connection_Template.md](./Connection_Template.md) | Blueprint for documenting the complex, qualitative relationship between two specific Archetypes (the core "Mandala" connections). |
| [Procedural_Template.md](./Procedural_Template.md) | Step-by-step guide for common repository tasks (e.g., adding data, generating files). |
