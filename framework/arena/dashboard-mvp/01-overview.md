# The Regenerative Capital Dashboard: Overview

## Purpose & Vision

The Regenerative Capital Dashboard is a **cybernetic tool for conscious transition**. It moves beyond measuring mere financial transaction (GDP) to track the holistic vitality of a community or bioregion. Its core purpose is to make visible the health of the systems that enable true flourishing, guided by the principles of the **Solarpunk Mandala** and the **Conscious Regeneration Economy (CRE)**.

## Core Concept: The 5+1 Display

The MVP (Minimum Viable Product) dashboard is built around a "5+1" display model that balances simplicity with diagnostic power:

1.  **The Five Domain Scorecards:** These track health across the five core circles of community life, each analyzed through four specific types of capital (20 capitals total).
2.  **The One Diagnostic Visualization:** This synthesizes the scorecard data into an intuitive, actionable shape.

### 1. The Regeneration Flower (Primary Diagnostic)

The **Regeneration Flower** is a hexagonal radar chart where each vertex represents one of the five domains. The community's aggregate score for each domain plots a point, and connecting these points creates a unique shape.

*   **A small, lopsided shape** indicates a struggling, imbalanced system.
*   **A large, symmetrical shape** indicates robust, balanced regeneration.
*   **The communal goal** is to collaboratively "grow and round out the flower" with each assessment cycle.

## Implementation Rhythm: The 4-Week Sprint Cycle

The dashboard is designed to be used in a recurring, ritualistic cycle that turns measurement into learning and action.

| Week | Phase | Key Activities |
| :--- | :--- | :--- |
| **Week 1** | **Gather & Input** | Collect quantitative data from community logs and launch the qualitative [Community Poll](./04-poll-questions.md). |
| **Week 2** | **Calculate & Visualize** | Process data, calculate domain scores and Œ±, and update the Regeneration Flower visualization. |
| **Week 3** | **Ritual of Diagnosis** | The community gathers in a **Council of the Whole** to observe the Flower, reflect on narrative data, and ask: **"Which rigid pattern ('cube') must be dissolved to allow greater flow?"** |
| **Week 4** | **Prescribe & Act** | Based on the diagnosis, the council selects a protocol from the Mandala's appendices to experiment with in the next cycle (e.g., a Right Livelihood Audit, a new communal ritual). |

*For a complete guide to this process, see [05-implementation-guide.md](./05-implementation-guide.md).*
